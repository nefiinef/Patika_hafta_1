# Tablolarla Çalışmak
 Bu proje temel HTML tablolarını öğrenmek için oluşturulmuştur.

 Temel olarak:
- `<table>`, `<tr>`, `<td>` ve `<th>` etiketleri kullanıldı.
- Basit stil ve renk uygulamaları eklendi.
## Kullanılan Teknolojiler
- HTML5
- CSS
